📝 แบบฟอร์มลงทะเบียน

✨ ระบบลงทะเบียนอัตโนมัติ (Auto Sync)

กรอกข้อมูลและส่งใบลงทะเบียนได้ทันที ระบบจะบันทึกข้อมูลลง Google Sheets โดยอัตโนมัติ และซิงค์ข้อมูลทุกครั้งที่เปิดเว็บไซต์ พร้อมใช้งานทันที!

👨‍💼 เข้าสู่ระบบผู้ดูแล

👨‍💼 แผงควบคุมผู้ดูแลระบบ

📈 ภาพรวมระบบ (พร้อมใช้งาน)

แสดงสถิติและข้อมูลสำคัญของระบบลงทะเบียน ระบบเชื่อมต่อกับ Google Sheets พร้อมใช้งาน Auto Sync ทันที!

🔗 การเชื่อมต่อ Google Apps Script

✅ ระบบพร้อมใช้งาน!
Google Apps Script URL ได้ถูกฝังไว้ในโค้ดแล้ว คุณสามารถใช้งานระบบได้ทันทีโดยไม่ต้องตั้งค่าเพิ่มเติม
💡 URL เริ่มต้นถูกฝังไว้ในโค้ดแล้ว หากต้องการเปลี่ยน URL ใหม่ สามารถใส่ที่นี่ได้

🤖 Auto Sync Settings

เมื่อเปิดใช้งาน ระบบจะซิงค์ข้อมูลจาก Google Sheets อัตโนมัติทุกครั้งที่มีผู้เปิดเว็บไซต์

🧪 ทดสอบระบบ

💡 คำแนะนำ: ทดสอบตามลำดับ: เชื่อมต่อ → ส่งข้อมูล → ระบบ (ปุ่มรับข้อมูลเป็นตัวเลือกเสริม)

📋 ข้อมูลการตั้งค่า

🎉 ระบบพร้อมใช้งานแล้ว!

✅ Google Apps Script URL: ฝังไว้ในโค้ดแล้ว

✅ Auto Sync: เปิดใช้งานโดยค่าเริ่มต้น

✅ การเชื่อมต่อ: ทำงานอัตโนมัติ


🛠️ หากต้องการเปลี่ยนแปลง:

  1. สร้าง Google Apps Script ใหม่ที่ script.google.com
  2. วางโค้ด Google Apps Script (ถามผู้พัฒนาได้)
  3. Deploy และคัดลอก URL ใหม่
  4. ใส่ URL ใหม่ในช่องด้านบนและบันทึก
  5. ทดสอบการเชื่อมต่อ

🔐 การจัดการระบบ
