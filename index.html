🎓 ระบบลงทะเบียนนักเรียน

ระบบดิจิทัลสำหรับการลงทะเบียนเก็บคะแนนแบบออนไลน์

📊 ตรวจสอบสถานะการลงทะเบียน

👨‍💼 เข้าสู่ระบบผู้ดูแล

👨‍💼 แผงควบคุมผู้ดูแลระบบ

📈 ภาพรวมระบบ

แสดงสถิติและข้อมูลสำคัญของระบบลงทะเบียน รวมถึงสถานะการเชื่อมต่อกับ Google Sheets

🔗 การเชื่อมต่อ Google Apps Script

URL ต้องลงท้ายด้วย /exec และต้องเป็น URL ที่ Deploy แล้ว

🧪 ทดสอบระบบ

💡 คำแนะนำ: ทดสอบตามลำดับ: เชื่อมต่อ → ส่งข้อมูล → ระบบ (ปุ่มรับข้อมูลเป็นตัวเลือกเสริม)

📋 คำแนะนำการตั้งค่า

วิธีการตั้งค่า Google Apps Script

  1. สร้าง Google Apps Script: ไปที่ script.google.com
  2. วางโค้ด: วางโค้ด Google Apps Script ที่ให้ไว้
  3. Deploy: กด Deploy > New Deployment
  4. ตั้งค่าสิทธิ์: Execute as: Me และ Who has access: Anyone
  5. คัดลอก URL: คัดลอก Web app URL มาใส่ด้านบน
  6. ทดสอบ: กดปุ่มทดสอบเพื่อตรวจสอบ

🔐 การจัดการระบบ
